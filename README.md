# PolicyPal ✦

An elegant AI-powered HR policy assistant that lets employees and HR teams upload any policy document and ask questions in plain language — no more hunting through pages of policy text.

## What it does

- **PDF & Text Support** — Upload any HR policy document in PDF or .txt format
- **Plain Language Q&A** — Ask anything about leave, benefits, remote work, grievances, performance reviews — in plain conversational language
- **Instant Answers** — Get precise, clear answers pulled directly from your policy document
- **Suggested Questions** — Pre-loaded common HR questions to get you started instantly
- **Conversation Memory** — Ask follow-up questions naturally, just like talking to an HR manager

## Built with

- React
- Anthropic Claude API (claude-sonnet-4-5)
- PDF.js (for PDF parsing)

## How to run locally

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file in the root with: `REACT_APP_API_KEY=your-anthropic-key`
4. Run `npm start`
5. Open `http://localhost:3000`
6. Upload any HR policy PDF or text file and start asking questions
